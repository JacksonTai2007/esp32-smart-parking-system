# 硬件接线说明

引脚总表见 docs/pinout.md（代码权威定义：`config/Pins.h`）。
本文按模块给出具体接线、供电与排查要点。**接线前务必通读 pinout.md 的注意事项。**

> 本期方案：红外车位传感器做车辆识别 + 车位管理 + 按时长计费，并加入 SG90
> 出入口道闸（可由 `ENABLE_GATE` 关闭）。放行只靠红外识别车辆，**不做 RFID 刷卡**；
> HC-SR04 入口超声波未纳入。
> 模块少，USB 5V 一般可直接供电；舵机抖动时给舵机单独 5V 并共地。

## 供电架构

```
USB 5V ──► ESP32 DevKit（板载稳压出 3.3V）
                │ 3.3V ──► OLED、红外车位模块(3.3~5V 均可)、蜂鸣器模块
                │ 5V(VIN) ──► SG90 道闸舵机（电流较大；抖动/复位时改独立 5V）
                │
                └──► 所有模块 GND 必须与 ESP32 GND 连通（共地！）
```

- 仅一个 SG90 舵机时通常 USB 供电即可；若舵机动作时 ESP32 复位或舵机抖动，
  给舵机单独接 5V 电源，并与 ESP32 **共地**
- **所有地必须连通**：ESP32 GND 与各模块 GND 接在同一排

## 1. 红外车位传感器 × 2~4（车辆识别核心）

| 模块引脚 | 接到 |
| --- | --- |
| VCC | 3.3V（多数模块 3.3~5V 兼容） |
| GND | GND |
| OUT | 车位1→GPIO 32，车位2→GPIO 33，车位3→GPIO 34，车位4→GPIO 35 |

- 只接 2~3 个车位时，把 `Settings.h` 的 `TOTAL_PARKING_SLOTS` 改成实际数量，
  按顺序占用 32→33→34 引脚
- 常见红外避障模块**检测到物体输出 LOW**（板载 LED 亮）；
  若你的模块相反，把 `SLOT_OCCUPIED_LEVEL` 改为 `HIGH`
- 模块上的电位器可调检测距离，演示前调到稳定触发小车模型的距离
- 本系统把"车位被占用"当作"识别到一辆车"：占用瞬间开始计时，
  恢复空闲瞬间结算费用

## 2. OLED SSD1306（I2C）

| OLED 引脚 | 接到 |
| --- | --- |
| VCC | 3.3V |
| GND | GND |
| SDA | GPIO 21 |
| SCL | GPIO 22 |

不亮时排查：I2C 地址多数为 0x3C，少数 0x3D（改 `Settings.h` 的
`OLED_I2C_ADDRESS`）；可先跑 I2C scanner 示例确认。

## 3. 蜂鸣器

| 模块引脚 | 接到 |
| --- | --- |
| VCC | 3.3V |
| GND | GND |
| I/O | GPIO 14 |

默认按"高电平响"的**有源蜂鸣器**模块处理；若是低电平触发，
改 `Settings.h` 的 `BUZZER_ACTIVE_LEVEL = LOW`。无源蜂鸣器需 PWM 驱动，
本期不支持，购买时认准"有源/active buzzer"。

提示音节奏：入场 1 短响、离场（已计费）2 短响、满位 1 长响。

## 4. SG90 出入口道闸（舵机）

| 舵机线 | 接到 |
| --- | --- |
| 橙（信号） | GPIO 13 |
| 红（VCC） | 5V（VIN；抖动/复位则独立 5V 供电） |
| 棕（GND） | GND（与 ESP32 共地） |

- 由 ESP32 LEDC 生成 50Hz PWM 驱动，**不需第三方库**；上电即落杆。
- 开 / 合角度与抬杆保持时长在 `Settings.h`：`GATE_SERVO_OPEN_DEG` /
  `GATE_SERVO_CLOSED_DEG` / `GATE_OPEN_HOLD_MS`；方向反了把两个角度对调。
- 闸杆用卡纸条粘在舵机摆臂上；进 / 出场自动抬杆、数秒后自动落杆，
  网页另有「开闸 / 落闸」手动按钮。
- 不需要道闸时置 `ENABLE_GATE 0`，相关代码退化为空实现。

## 上电自检顺序（首次接线后）

1. 只接 ESP32 + OLED → 烧录 → 看 OLED 出画面、串口出 banner
2. 加蜂鸣器 → 遮挡/移开车位传感器应分别听到入场 1 短响、离场 2 短响
3. 加红外车位 → 逐个遮挡看车位日志（占用/空闲翻转、剩余车位变化）
4. 加 SG90 道闸 → 串口看 `[Gate] init OK`（上电落杆）；进出场时应抬杆、
   数秒后自动落杆；网页点「开闸/落闸」舵机应响应
5. 完整流程：遮挡某车位 → 道闸抬杆 + 入场计时 → 等待几秒 → 移开 → 看串口/OLED/网页
   显示本次停留时长与费用、累计收入累加

完整测试清单见 tests/README.md。
